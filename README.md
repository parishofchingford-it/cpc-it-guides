# CPC IT Guides

Public-facing Markdown guides and operational workflows for the Parish of Chingford.

This repository is intended primarily for:

* user guides
* publishing workflows
* operational documentation

The repository is designed so selected guides can be published automatically to the Parish website.

## Structure

* `guides/` - public-facing guides and documentation
* `guides/<guide-name>` - each guide in its own directory alongside its associated media

## Principles

* Markdown is the source of truth
* Git provides version history and operational continuity
* Public documentation is separated from internal operational/audit information
* Documentation should remain understandable and maintainable by non-technical users where possible
* WordPress plugin, e.g. `Git to Write` used to automatically publish to the website
