# Guide to Posting Pews News

This guide explains the standard process for publishing the weekly Pews News post on the Parish website.

The aim is to create a simple, clear post containing:

* the weekly Pews News PDF
* a short introductory paragraph
* the featured image
* the correct categories and metadata

Once published, the post link should also be shared to the parish WhatsApp channel.

---
## Video walkthrough

A video walkthrough (4 minutes) of these instructions is available here:

https://www.youtube.com/watch?v=uuSB4HFHLUk

The detailed written steps below remain the primary reference, but the video may help if you are unfamiliar with the WordPress editor.

---

# 1. Upload the PDF

1. Log into the <a href="https://parishofchingford.org.uk/wp-admin/" target="WP">website's WordPress editor</a> 
2. Go to <a href="https://parishofchingford.org.uk/wp-admin/media-new.php" target="WP">Media → Add New Media File</a>.
3. Click **Select Files**
4. Select the new Pews News PDF.

---

# 2. Create the post

1. Go to <a href="https://parishofchingford.org.uk/wp-admin/post-new.php" target="WP">Posts → Add New Post</a>.
2. Set the post title using the format: ```Pews News – Sunday 14 June 2026 – Second Sunday after Trinity```

Adjust the date and liturgical occasion as appropriate.

---

# 3. Set the featured image

In the right-hand settings panel:

1. Click **Set Featured image**.
2. In the media library, search for: ```Pews News Featured Image```
3. Click **Set featured image**.

---

# 4. Set the excerpt

Still in the right-hand settings panel:

1. Click **Add an excerpt...**.
2. Enter a short description such as: ```Pews News and parish notices for Sunday 14 June 2026.```

Adjust for the correct date as appropriate. This does not need to be elaborate but you can if you wish, e.g. 
mentioning a key event, such as Hilary's announcement. The excerpt can  be kept quite short. 

---

# 5. Set categories

Still in the right-hand settings panel:

1. Open the **Categories** section.
2. Tick:
    * ```Community```
    * ```Pews News```
3. Set the **Primary category** to: ```Pews News```

---

# 6. Add the introductory paragraph and PDF link

In the main editor area, add a paragraph such as: ```Here are the main parish notices from Pews News for Sunday 
14 June 2026. You can also download the full PDF.```

Then create a link to the PDF: select the words: ```download the full PDF```

and select the ***create link*** button - looks like a chain link

1. In the search box type the name of the PDF, e.g. `Pews News 2026-06-14.pdf`
2. Press Enter
3. Make the PDF open in a new tab, by
   1. Clicking the link you just created
   2. Clicking the little pencil icon
   3. Scrolling down to "Advanced"
   4. Check "Open in new tab"
   5. Click on "Apply" button.

---

# 7. Embed the PDF

1. Add a new block underneath the introductory paragraph.
2. Add a "PDF block", type `/PDF Embedder`
3. In the RHS panel press **Select PDF**
4. Click **Media Library** tab
4. Select this weeks PDF
5. Click the **Select** button

---

# 8. Set Yoast SEO metadata

Scroll down to the **Yoast SEO** section underneath the editor.

Set the Meta description to something simple such as:```Pews News and parish notices for Sunday 14 June 2026.```

This helps search engines and social media previews.

---

# 9. Publish the post

1. Click the **Publish** button at the top right.
2. Review the publishing options if prompted.
3. Click the final **Publish** confirmation button.
4. Click **View Post**.
5. Check the post looks ok. You can still edit it even after it is published.
6. Copy the post URL.

---

# 10. Share to WhatsApp

1. Post the new Pews News link to the parish WhatsApp channel.
2. Just paste the link - no need to include a description as WhatsApp will use the excerpt you typed before.

# Optional additional sharing

Optionally, the post can also be shared to the parish Facebook page with a short accompanying message.
